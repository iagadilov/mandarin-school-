# mandarin-school-

Mandarin Edu Center mental arithmetic trainer.

## Scope

Stage 1 web trainer:
- show numbers one by one;
- input final answer;
- automatic answer check;
- configurable rows, examples count, and speed;
- units 1-9 without formulas;
- tens 10-90, identical double digits 11-99, and three-digit numbers up to 999 by the same methodology;
- formulas on 5/10 by methodist rules.

This stage does not include student accounts, teacher cabinet, student database, reports, homework, payments, integrations, or mobile apps.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Stack

Next.js 16, React 19, Tailwind CSS v4, TypeScript.
